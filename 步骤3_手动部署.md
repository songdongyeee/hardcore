# 步骤 3: 手动部署后端钩子

由于 SSH 连接问题，我们改用手动部署。

## 方法 A: 通过宝塔面板（最简单）

### 1. 登录宝塔面板
- 打开宝塔面板
- 进入文件管理

### 2. 备份现有文件
导航到：`/www/pocketbase/pb_hooks/`

找到 `aliyun_asr.pb.js`，右键 → 重命名为：
```
aliyun_asr.pb.js.backup
```

### 3. 上传新文件
在同一目录：
1. 点击"上传"
2. 选择本地文件：
   `/Users/geo/Downloads/hardcoreEnglish/pb_hooks/dual_engine_asr.pb.js`
3. 上传完成后，重命名为：
   ```
   aliyun_asr.pb.js
   ```

### 4. 重启 PocketBase
在宝塔面板：
- PM2 管理器
- 找到 `pocketbase`
- 点击"重启"

---

## 方法 B: 使用 SSH 客户端（如 FinalShell）

### 1. 连接服务器
- Host: zjcnex.top
- User: root

### 2. 上传文件
通过 SFTP：
- 本地：`/Users/geo/Downloads/hardcoreEnglish/pb_hooks/dual_engine_asr.pb.js`
- 远程：`/www/pocketbase/pb_hooks/aliyun_asr.pb.js`

### 3. 重启
在终端执行：
```bash
pm2 restart pocketbase
```

---

## 方法 C: 使用 scp 命令（需要密码）

在你的 Mac 终端：

```bash
# 上传文件（会提示输入密码）
scp pb_hooks/dual_engine_asr.pb.js root@zjcnex.top:/www/pocketbase/pb_hooks/aliyun_asr.pb.js

# SSH 重启（会提示输入密码）
ssh root@zjcnex.top 'pm2 restart pocketbase'
```
