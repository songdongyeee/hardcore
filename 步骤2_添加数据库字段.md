### 2. 进入 Collections

### 3. 添加新字段
点击 "+ New field"，配置如下：

**字段类型**: Select

**配置项**:
```
Name: asr_engine
Options (选项值):
  - aliyun
  - whisper
Max select: 1
Presentable: 不勾选
Required: 不勾选
```

### 4. 保存
- 点击字段的 "Save"
- 点击 Collection 底部的 "Save collection"
